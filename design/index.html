Design Style

From now on, I want to use a consistent design style.

Core Principles

Everything uses tailwind.css , the input stylesheet is called style.css , and the output stylesheet is called site.css .

Everything is in dark mode ( bg-neutral-800, text-white ).

Meta Tags

I'll mostly use the template provided by VS Code . This means these tags:

  • <lang="en">
  • <meta charset="UTF-8">
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">

I'll also use a couple others:

  • <meta name="accent-color" content="#324AD5">
  • <meta name="description" content="[insert description here]">
  • <title>[insert title here]</title>
