Design Style

From now on, I want to use a consistent design style.

  • Use tailwind.css, with the typography library. Input files are style.css and output is site.css .
  • Use bg-neutral-800 and text-white for the body.
  • Also, change some meta tags from the default VS code template:
    • Remove the IE=edge meta tag, IE is outdated.
    • Add the accent-color meta tag, set it to #324AD5 (my color)
    • Add the description meta tag, set the content to whatever fits.

Specific examples

Tailwind config file:


module.exports = {
    content: ["**/*.html"],
    theme: {
        extend: {},
    },
    plugins: [require("@tailwindcss/typography")],
};
